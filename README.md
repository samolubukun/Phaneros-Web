# Phaneros Web

This is the web version of Phaneros, an intelligent scripture detection and presentation suite.

### Core Technology
- **Frontend**: React 19 with Zustand state management and RxJS signal orchestration.
- **Backend**: High-performance Rust (Axum) core with SQLite and HNSW similarity indexing.
- **STT Infrastructure**: Resilient proxy bridging Whisper High-Fidelity, Deepgram Nova-3, and Vosk Local fallback.
- **Intelligence**: Semantic search and vectorized allusions powered by Cloudflare Workers AI.

For the main web and desktop application, visit the core repository: [Phaneros Core](https://github.com/samolubukun/Phaneros/)
